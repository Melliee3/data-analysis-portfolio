# Data Analysis Portfolio

This repository contains Python-based data analysis projects developed as part of a Data Science Summer School and independent practice.

The projects focus on:
- Data cleaning and validation
- Exploratory data analysis (EDA)
- Feature engineering
- Data visualization using Python libraries

The goal of this portfolio is to demonstrate practical data analysis skills using real-world datasets across different domains.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly (basic)

## Projects
Each project is located in the `projects/` folder and includes:
- A Python script with the analysis
- A short README explaining the objective and key steps

## About
This portfolio is intended for entry-level and junior data-related roles, with emphasis on analytical thinking, clean code structure and meaningful visualizations.
