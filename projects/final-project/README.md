# Finance Liquor Sales Data Analysis

## Overview
This project focuses on the analysis of liquor sales data with an emphasis on financial performance, sales trends, and revenue patterns.  
The goal is to extract actionable insights through data cleaning, exploratory data analysis, and visualization.

The project was completed as the **Final Assignment** of a Data Science Summer School and demonstrates an end-to-end data analysis workflow using Python.

---

## Objectives
- Analyze sales and revenue trends over time
- Identify high-performing product categories and regions
- Explore seasonality and demand patterns
- Practice structured data analysis and insight communication

---

## Dataset
The dataset includes transactional liquor sales data with variables such as:
- Date of sale
- Product category
- Sales volume
- Revenue
- Location / region (if applicable)

*Note: The dataset is used for educational purposes.*
